# 학원 명단 시트 — 등록회차 클릭 → 회차 칸 자동 생성

명단 시트에 붙여서 **G(등록회차)를 고르면 그 자리에서 바로 회차 칸이 생성**되게 만드는 도구입니다.

설치하면 열 구조는 이렇게 됩니다:

`A 번호 · B 이름 · C 학교/학년 · D 휴대전화 · E 등록여부 · F 결제금액 · G 등록회차 · H 형제할인 · I 등록일 · J 다음등록일 · K~O 주차 띠(5주) · P~ 회차 칸`

---

## 동작 규칙

| 등록회차 선택 | 생성되는 칸 | 줄 수 |
|---|---|---|
| `주1회 60분 월` | 4칸 (분홍) | 1줄 |
| `주2회 60분 월` | 8칸 (분홍) | 1줄 |
| `주2회 90분 월` | 8칸 (초록) | 1줄 |
| `주2회 90분 분기` | 8칸씩 × 3줄 = 24칸 | **3줄 (아래로 2줄 자동 삽입)** |
| `주3회 120분 분기` | 12칸씩 × 3줄 = 36칸 | 3줄 |
| `매일반 90분` | 가득 (초록) × 3줄 | 3줄 |

- **칸 개수** = 주N회 × 4 (한 달치). 주1회=4 · 주2회=8 · 주3회=12.
- **분기납** = 아래로 2줄이 자동 삽입되어 총 3줄(월별 1줄). **월납** = 1줄.
- **색**: 60분 분홍 · 90분 초록 · 120분 노랑.

---

## 설치 (3분)

1. 명단 시트에서 **확장 프로그램 ▸ Apps Script**.
2. `Code.gs` 전체를 붙여넣고 저장(💾).
3. 시트로 돌아와 새로고침 → 상단 **📚 학원관리** 메뉴 생성.
4. **📚 학원관리 ▸ ① 설치 / 드롭다운 적용** 실행. (권한 승인 1회)
   - G열에 등록회차 드롭다운, E열에 등록여부 드롭다운(초록/노랑/빨강 색)이 적용됩니다.
   - `플랜단가` 시트가 없으면 자동 생성됩니다(금액 자동 입력용).

> ⚠️ 열 위치가 중요합니다: **E=등록여부, F=결제금액, G=등록회차, H열부터 회차 칸.**
> 현재 시트가 이 순서면 그대로 작동합니다.

---

## 사용법

### 1) 등록회차 선택 → 칸 자동 생성
- G열에서 `주2회 90분 분기` 처럼 고르면 → 아래로 2줄이 생기고 8칸씩 3줄이 색칠됩니다.
- 결제금액(F)이 비어 있으면 `플랜단가` 표에서 금액을 자동으로 채웁니다.
  (이미 금액이 있으면 건드리지 않습니다.)

### 2) 출석 체크 (두 가지)
- **방법 A** — 회차 칸에 온 날짜 직접 입력(예: `3/4`).
- **방법 B** — 칸 선택 후 **📚 학원관리 ▸ ✅ 선택 칸 오늘 출석 체크**.
- 날짜가 든 칸은 **회색(사용됨)** → 남은 색칸 = 남은 회차.
- 취소: **↩️ 선택 칸 출석 취소** 또는 칸 내용 삭제.

### 3) 형제할인 (H열 드롭다운)
- `형제할인` 칸은 **`정상` / `형제할인`** 드롭다운입니다.
- **`형제할인`** 을 고르면 결제금액(F)이 즉시 **5% 할인가**로 바뀝니다.
- 다시 **`정상`** 을 고르면 **원래 금액으로 복원**됩니다. (원가는 자동 보관)

### 3-1) 번호(A열)·구분선
- 이름을 입력하면 **번호가 자동으로 1, 2, 3…** 순서대로 매겨집니다(분기납 연속 줄 제외).
- 학생이 바뀌는 줄마다 **굵은 가로 구분선**이 그어져 한 명씩 구분됩니다.
- 틀어지면 **📚 학원관리 ▸ 🔢 번호·구분선 다시 정리**.

### 3-2) 등록일(I)·다음등록일(J) — 달력 + 색 경고
- 두 칸 모두 **더블클릭하면 달력**이 떠서 날짜를 고를 수 있습니다.
- 등록회차를 고르면 **등록일=오늘**, **다음등록일=등록일+개월(월납 1·분기 3)** 이 자동 입력됩니다(직접 고쳐도 됨).
- **다음등록일 칸 색 경고**(결제 챙기기용):
  - **3일 전 = 노랑** · **1일 전 = 주황** · **당일 또는 지남 = 진한 빨강**

### 4) 주차 띠 (K~O열) — 한 주라도 안 왔는지 한눈에
- 등록회차를 고르면 **등록일(I)이 오늘 날짜로 자동 기록**됩니다.
- **한 줄 = 한 달** 구조라, 분기납은 **3줄**로 회차 칸과 나란히 정렬됩니다.
- 주는 **달력의 월~일 주(월요일 시작)** 기준이며, **등록 주부터 첫 칸에 차례로**
  1주·2주·…·15주(3줄×5)까지 채워집니다. 월 중간에 시작해도 3줄로 분기 전체가 들어갑니다.
- 각 주 칸에는 **그 주 출석 횟수**가 표시됩니다.
  - 지난 주인데 한 번도 안 왔으면 **빨강(0)**, 왔으면 **초록**, 등록 전·미래 주는 빈칸.
- 시간이 지나면 **📚 학원관리 ▸ 🔄 주차 띠 전체 새로고침 (오늘 기준)** 으로 갱신하세요.
  (출석 데이터는 보존됩니다.)

### 4-1) 등록회차 무지개색
- 등록회차 21개 옵션마다 **무지개색**이 차례대로 들어가, **고르면 그 칸이 해당 색**으로 칠해집니다.
  (드롭다운 목록 팝업의 칩 색은 구글 정책상 코드로 못 바꿔, 선택된 칸 색으로 표시합니다.)

### 4-2) 60분 옵션을 빼고 싶을 때
- `Code.gs`의 `planOptions_()`에서 **`'60분'`을 목록에서 빼고** 저장 → `[학원관리 ▸ ① 설치]` 다시 실행.
- **이미 60분으로 등록된 학생은 그대로 유지**되고, 가격표·색상·계산도 60분을 계속 인식합니다.
  (새로 고를 때만 60분이 안 보일 뿐, 다른 기능에 전혀 지장 없습니다.)

### 5) 금액표 — `플랜단가` 시트
- **책나무 공식 교육비표**가 입력되어 있습니다 (유치부 60분 / 초등부 90분 / 중등 120분).
- 등록회차를 고르면 이 표에서 **정상금액이 자동 입력**됩니다. (금액이 비어있을 때만)
- 단가가 바뀌면 이 시트의 숫자만 수정하면 됩니다.

---

## 주의사항
- **등록회차는 학생의 첫 줄에서만** 고르세요. 분기납으로 생긴 연속 2줄(회색 표시)에서는 작동하지 않습니다.
- 등록회차를 **월납↔분기납으로 바꾸면 줄 수가 자동 조정**됩니다.
  분기→월로 바꾸면 아래 2줄이 삭제되므로, 그 줄의 출석 기록도 함께 사라집니다.
- 시간/횟수를 바꾸면 회차 칸이 새로 그려져 기존 출석 표시가 초기화됩니다.
- 한 줄 안의 회차 칸은 최대 31칸까지 지원합니다(매일반 3줄 = 최대 93칸).
