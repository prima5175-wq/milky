# 학원 명단 시트 자동화 (구글시트 + Apps Script)

명단 시트의 **G(등록회차)** 를 고르면 그 자리에서 **회차 칸이 자동 생성**됩니다.
주N회×4칸, 분기납이면 아래로 2줄 자동 삽입(총 3줄)·월납이면 1줄, 시간별 색칠
(60분 분홍·90분 초록·120분 노랑), 결제금액 자동 입력까지 됩니다.

시트 열 구조: `A 테스트날짜 · B 이름 · C 학교/학년 · D 휴대전화 · E 등록여부 · F 결제금액 · G 등록회차 · H~ 회차칸`

## 파일
- `Code.gs` — 구글시트 Apps Script 코드 (이 파일을 시트의 Apps Script에 붙여넣어 사용).
- `appsscript.json` — Apps Script 매니페스트(시간대 Asia/Seoul).
- `사용설명서.md` — 설치 및 사용법(한국어).
- `migrate.py` — 기존 엑셀을 새 양식으로 자동 이관하는 스크립트.

## 기존 데이터 이관
```
python3 migrate.py "기존시트.xlsx" "학원관리_이관본.xlsx"
```
- 신원·출석 날짜는 그대로 옮기고, 회차/시간/납부/금액/기간은 best-effort 파싱합니다.
- 원본 값은 손실 없이 `결제메모` 칸에 보존하며, 모호한 경우만 `⚠️검수`로 표시합니다.
- 출석은 **현재 등록 기간(결제일−개월) 이후**만 표시해 총회차·남은회차가 플랜과 일치합니다.
- 결과 `.xlsx`는 학생 개인정보를 포함하므로 git에 커밋하지 않습니다(`.gitignore`).
- 구글시트로 가져오기(파일 ▸ 가져오기) 후 `Code.gs`를 붙이면 자동화가 켜집니다.

## 빠른 시작
새 구글시트 → 확장 프로그램 ▸ Apps Script → `Code.gs` 붙여넣기 → 저장 →
시트로 돌아와 **📚 학원관리 ▸ ① 시트 초기화/설치** 실행.

자세한 내용은 [`사용설명서.md`](./사용설명서.md) 참고.
