# BookSpec 랜딩페이지 계획서

**목적**: GitHub Pages를 통해 BookSpec 프로젝트를 소개하는 정적 랜딩페이지
**출력**: `landing-page/index.html` (단일 파일)

---

## 섹션 구조

```
[Hero]       태그라인 + 핵심 가치 한 문장 + CTA
[Problem]    학부모의 Pain을 3개 카드로 공감 유도
[Solution]   BookSpec이 해결하는 방식 — 3가지 핵심 기능
[How it works] 학생 / 학부모 / 원장 플로우
[Features]   기능 그리드 (MVP 기준 7개)
[Social Proof] 시장 수치 + 페르소나 인용구
[Pricing]    학부모 플랜 / 원장 플랜
[CTA Footer] 베타 신청 폼 + 파트너 문의 링크
```

## 디자인 방향

- 테마: 딥스페이스 에디토리얼
- 배경: #080C1A (심우주 네이비)
- 액센트: #F5C842 (보이저 골드) + #4DFFD2 (오로라 민트)
- 헤드라인: Playfair Display / 본문: DM Sans
- 태그라인: "질문이 스펙이다"

## 기술 스펙

- Vanilla HTML/CSS/JS 단일 파일
- Google Fonts CDN
- Canvas API 파티클 배경
- IntersectionObserver 스크롤 애니메이션
- GitHub Pages 배포
