# 핸드폰 뒷자리 출석 체크인 설정 방법

아이가 태블릿에서 핸드폰 번호 뒷자리 4자리를 누르면, 오늘자 수업일지에 이름과 출석시간이
자동으로 기록되는 기능입니다. 하원시간은 기존에 세팅된 수식이 그대로 계산합니다.

지점(수업일지 구글시트)마다 아래 과정을 한 번씩 해주시면 됩니다.

## 1. 수강생명단 탭 만들기

해당 지점의 "수업일지" 구글시트에 새 탭을 하나 추가하고 이름을 **수강생명단** 으로
짓습니다. 컬럼은 아래처럼 딱 3개만 있으면 됩니다.

| A (번호) | B (이름) | C (전화번호) |
|---|---|---|
| 1 | 고은 | 010-9622-4570 |
| 2 | 김라희 | 010-8731-9338 |

- 형제/자매가 같은 번호를 쓰면 각자 한 줄씩, 전화번호만 똑같이 넣으면 됩니다.
- 신규 등록/퇴원 시 이 탭만 관리하면 되고, 기존의 복잡한 수강생 대장 시트는 건드리지
  않아도 됩니다.

## 2. Apps Script 붙여넣기

1. 수업일지 구글시트 상단 메뉴에서 **확장 프로그램 → Apps Script** 를 클릭합니다.
2. 기본으로 열려 있는 `Code.gs` 안의 내용을 전부 지우고, 이 저장소의
   `apps-script/CheckinScript.gs` 파일 내용을 그대로 붙여넣습니다.
3. 코드 맨 위 `SECRET_TOKEN = 'CHANGE_ME'` 부분을 원하는 문자열로 바꿉니다.
   (지점마다 다르게 설정하면 더 안전합니다. 예: `daechi2026`)
4. 저장(Ctrl+S) 합니다.

## 3. 웹앱으로 배포

1. Apps Script 화면 우측 상단 **배포 → 새 배포** 를 클릭합니다.
2. 유형 선택에서 **웹앱**을 선택합니다.
3. "실행할 사용자"는 **나**, "액세스 권한이 있는 사용자"는 **전체 허용**으로 설정합니다.
4. **배포**를 누르고, 권한 승인 요청이 뜨면 본인 구글 계정으로 승인합니다.
5. 배포 완료 후 나오는 **웹앱 URL**(`https://script.google.com/macros/s/.../exec` 형태)을
   복사해둡니다.

> 시트 구조를 나중에 수정하면 **배포 → 배포 관리 → 수정(연필 아이콘) → 새 버전** 으로
> 다시 배포해야 변경사항이 반영됩니다.

## 4. 태블릿에서 연결

1. 태블릿 브라우저로 `docs/checkin.html` (GitHub Pages로 배포된 주소)를 엽니다.
2. 처음 열면 설정창이 뜹니다. (다시 열고 싶으면 화면을 두 번 탭)
3. **Apps Script 웹앱 URL**에 3단계에서 복사한 주소를,
   **토큰**에 2단계에서 정한 `SECRET_TOKEN` 값을 입력하고 저장합니다.
4. 이제 뒷자리 4자리를 누르면 바로 출석 처리가 됩니다. 형제처럼 같은 번호를 쓰는 경우
   해당하는 이름들이 버튼으로 나타나고, 탭하면 그 학생만 체크인됩니다.

## 참고

- 오늘 날짜 탭(예: `7/17 금`, `7/17 (금)`, `7/20(월)` 등 표기는 지점마다 달라도 됩니다.
  "월/일"로 시작하는 탭을 자동으로 찾습니다)이 아직 만들어져 있지 않으면 체크인이 되지
  않고 안내 메시지가 뜹니다. 평소처럼 오늘자 탭을 미리 만들어두면 됩니다.
- 하루 탭에서 "이름" / "시작시간" / "종료시간" 이 적힌 헤더 칸을 자동으로 찾아서 그
  위치에 맞게 기록합니다. 지점마다 컬럼 순서가 달라도(대치점은 이름이 C열, 구룡초는
  이름이 다른 열인 식으로 달라도) 컬럼 번호를 따로 맞춰줄 필요가 없습니다. 다만 헤더
  칸의 문구 자체를 "이름"/"시작시간"/"종료시간" 이 아닌 다른 표현으로 쓰는 지점이 있다면
  `CheckinScript.gs` 상단의 `*_HEADER_CANDIDATES` 배열에 그 표현을 추가해주세요.
- 이미 출석 처리된 학생이 다시 누르면 "이미 출석 처리되었어요" 라고만 안내하고,
  기존 기록은 덮어쓰지 않습니다.
- 지점마다 배포되는 웹앱 URL과 토큰이 다르므로, 지점 태블릿마다 3~4단계를 각각
  진행해야 합니다.
- "수강생명단" 탭에 전화번호가 비어 있는 학생은 조회가 되지 않습니다. 대장에 번호가
  아직 없는 학생은 먼저 번호를 채워주세요.
