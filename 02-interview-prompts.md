# BookSpec 사용자 인터뷰 프롬프트

**서비스**: 학부모 연동 독서 보상 결제 시스템  
**시장 매력도**: 21 / 25점  
**선택 이유**: 결제자(학부모)와 사용자(자녀)가 분리된 구조, 학원 네트워크를 통한 초기 유저 획득 비용 거의 0

---

## 타겟 그룹 1: 독서논술학원 수강 초등 학부모 (강남·서초·대치)

**페르소나**
- 이름: 박수진
- 프로필: 42세, 서초구 전업주부, 초등 4학년 딸
- 시도한 서비스: 종이 독서록 + 카카오톡 단톡방
- 사용 기간: 종이 독서록 6개월

**인터뷰 프롬프트**
```
42세 전업주부 수진씨는 딸 서연이를 독서논술학원에 보내고 있어. 월 18만원이나 내는데
딸이 독서록을 제대로 쓰는지 확인할 방법이 없어. 학원에서 준 종이 독서록은 있는데
딸이 방에서 혼자 쓰고 나오니까 대충 쓰는지 진지하게 쓰는지 알 수가 없고,
선생님한테 카톡 보내는 것도 민망해서 못 물어보고 있어.

수진씨 입장에서, 독서록 관리 문제로 어떻게 고생했는지 구체적으로 이야기해줘:
- 처음에 딸한테 독서록 쓰라고 했을 때 반응이 어땠어?
- 제대로 쓰는지 확인하려고 시도해본 방법이 있어?
- 어느 순간 포기하거나 '에라 모르겠다' 하게 됐어?
- 학원에 뭔가 물어보거나 요청하기 망설여지는 이유가 뭐야?
- 딸이 독서록을 쓸 때마다 네가 직접 포인트를 줄 수 있다면 어떨 것 같아?
- 사실 어떤 기능이 있었으면 6개월 전부터 계속 썼을 것 같아?
```

## 타겟 그룹 2: 학원비 민감한 수도권 외 초등 학부모

**페르소나**
- 이름: 이미경
- 프로필: 38세, 경기도 평택 간호사 3교대, 초등 3학년 아들
- 시도한 서비스: 도서관 독서통장, 스티커 보상표 (직접 제작)
- 사용 기간: 독서통장 2주, 스티커 표 한 달

**인터뷰 프롬프트**
```
38세 간호사 미경씨는 3교대 근무 중에도 아들 준호 독서 습관 만들려고 도서관 독서통장도
해보고 집에서 스티커 칸 만들어서 붙여줬어. 처음 일주일은 준호가 신나서 책도 읽고 스티커도
붙였는데, 2주 차부터 "엄마 이거 언제 끝나?" 하더니 한 달째에는 스티커 표가 반도 안 찼어.

미경씨 입장에서, 아이 독서 보상 시스템이 실패한 경험을 구체적으로 이야기해줘:
- 처음에 어떤 마음으로 시작했어?
- 준호가 흥미를 잃기 시작한 게 언제부터야?
- 스티커나 독서통장이 왜 효과가 없었던 것 같아?
- 네가 교대 근무라 없는 날 아이 독서 관리가 어떻게 돼?
- 아이가 책 읽으면 실제로 쓸 수 있는 포인트가 생기고 그걸 네가 직접 쏴줄 수 있다면 달랐을까?
- 지금 형편에서 월 얼마면 납득하고 쓸 것 같아?
```

## 타겟 그룹 3: 중학생 세특·포트폴리오 관리 학부모

**페르소나**
- 이름: 정혜영
- 프로필: 46세, 강남구 카페 사장, 중학교 2학년 아들
- 시도한 서비스: 수기 독서 노트, 네이버 메모 앱
- 사용 기간: 독서 노트 3일, 네이버 메모 1주일

**인터뷰 프롬프트**
```
46세 카페 사장 혜영씨는 입시 유튜브 보다가 세특에 독서 기록이 들어간다는 걸 알게 됐어.
아들 민준이한테 "책 읽고 생각 정리해서 노트에 써"라고 했는데, 민준이는 사흘 쓰더니
"엄마 이거 왜 해?" 하고 던졌어. 네이버 메모에 써보라고 했더니 일주일 만에 아무것도 없어.

혜영씨 입장에서, 독서 기록이 세특에 연결 안 되는 답답함을 이야기해줘:
- 입시에서 독서가 중요하다는 걸 처음 알았을 때 어떤 기분이었어?
- 민준이한테 노트 쓰라고 했을 때 왜 안 됐다고 생각해?
- AI가 민준이 독서 내용을 요약하고 "이 부분이 생명과학 세특에 쓸 수 있어요"라고 알려주면 쓸 것 같아?
- 이걸 위해 월 얼마면 낼 의향이 있어?
```

## 타겟 그룹 4: 독서논술 학원·공부방 원장 (B2B)

**페르소나**
- 이름: 강은주
- 프로필: 49세, 노원구 독서논술 공부방, 수강생 15명 운영 7년차
- 시도한 서비스: 구글 스프레드시트, 카카오톡 채널
- 사용 기간: 구글 시트 3개월째 사용 중이나 업데이트 자주 빠짐

**인터뷰 프롬프트**
```
49세 공부방 원장 은주씨는 수강생 15명 독서 현황을 구글 시트에 관리하고 있어.
3개월째 접어드니 입력 빠뜨리는 아이들이 생기고 학부모한테 "이번 달 우리 아이가 어떤 책
읽었나요?" 카톡 오면 그때마다 시트 뒤져서 일일이 타이핑해서 보내고 있어.

은주씨 입장에서, 수강생 독서 관리 시스템이 어떻게 망가지고 있는지 이야기해줘:
- 구글 시트 처음 만들었을 때 어떤 기대를 했어?
- 어느 시점부터 관리가 안 되기 시작했어?
- 학부모한테 현황 보고할 때 가장 힘든 게 뭐야?
- 수강생이 직접 앱에 독서록을 쓰고, 학부모가 그걸 바로 볼 수 있다면 공부방 운영이 어떻게 달라질 것 같아?
- 공부방 입장에서 월 얼마, 혹은 수강생 1인당 얼마까지 낼 수 있어?
```

## 타겟 그룹 5: 공교육 교사 (독서교육 담당)

**페르소나**
- 이름: 윤재성
- 프로필: 34세, 인천 초등학교 4학년 담임, 교직 7년차
- 시도한 서비스: 교육부 독서로 시스템, 학급 밴드 독서 인증
- 사용 기간: 독서로 1학기 후 포기, 밴드 인증 2주

**인터뷰 프롬프트**
```
34세 초등 담임 재성씨는 아이들 독서 수업마다 교육부 독서로 시스템을 쓰려고 했어.
1학기 동안 써봤는데 아이들 25명 중에 로그인 제대로 하는 애가 절반도 안 됐고,
결국 2학기엔 그냥 종이 독서록으로 돌아갔어.

재성씨 입장에서, 학생 독서 기록 시스템이 왜 실패했는지 이야기해줘:
- 독서로 처음 썼을 때 기대가 있었어?
- 아이들이 로그인도 못 할 때 어떤 기분이었어?
- 결국 종이로 돌아가기로 결정한 순간이 있었어?
- 학부모한테 독서 현황 실시간 공개 요청 받았을 때 어떻게 대응했어?
- 학교 예산으로 구매하거나 교육청 사업으로 연계하는 방식이면 현실적으로 가능해?
```
