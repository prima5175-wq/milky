# 강사 시간표 자동화 시스템 (대치점 · 도곡점 · 구룡초점) — 30분 단위

**`입력` 시트에 강사 일정 한 줄만 적으면 → 정규/방학 시간표(30분 단위)·데일리현황·급여·대시보드가 전부 자동으로 채워지는** Google Sheets 시스템입니다.
`AppsScript_시간표.gs` 하나만 붙여넣고 실행하면 모든 탭이 생성됩니다.

- 시간표는 **30분 단위**라 3시반(15:30)·7시반(19:30) 같은 반시간도 정확히 표시됩니다.
- **재생성(다시 실행)해도** `강사_급여`의 이름·시급·월급과 `입력` 시트 일정은 **그대로 보존**됩니다.
- 강사 이름은 `강사_급여` 탭의 노란 "강사명" 칸에 적으면 모든 탭에 자동 반영됩니다.

## 설치 (3분)
1. 새 Google 스프레드시트 생성
2. **확장 프로그램 → Apps Script**
3. 기존 코드 지우고 `AppsScript_시간표.gs` 전체 붙여넣기 → 저장
4. 함수목록 **buildAll** 선택 → ▶ 실행 → 최초 1회 권한 허용
5. 이후엔 시트 상단 **[강사시간표] → [전체 다시 생성]**

## 입력 방법 (핵심)
`입력` 탭에 한 줄씩 적습니다. **요일은 체크박스로 클릭**하고 시간만 적으면 됩니다. 적는 즉시 시간표에 반영됩니다.

| 강사명 | 지점 | 구분 | 월 | 화 | 수 | 목 | 금 | 토 | 일 | 시작 | 종료 |
|---|---|---|---|---|---|---|---|---|---|---|---|
| 이도연 | 대치점 | 정규 | ☑ | ☑ | ☑ | ☑ | ☑ |  |  | 13 | 21 |
| 정호암 | 대치점 | 정규 | ☑ | ☑ |  |  | ☑ |  |  | 14 | 18 |
| 정호암 | 대치점 | 정규 |  |  |  |  |  | ☑ |  | 10 | 15.5 |
| 정호암 | 대치점 | 방학 | ☑ | ☑ |  |  | ☑ |  |  | 9 | 15 |

- **요일**: 해당 요일 칸을 **클릭(체크)** — 일일이 타이핑할 필요 없음
- **시간**: 24시간 숫자: 오후1시=`13`, 3시반=`15.5`, 7시반=`19.5`, 저녁9시=`21`, 10시반=`10.5`
- 한 강사가 **요일마다 시간이 다르면 줄을 나눠** 입력 (정호암 예시처럼)
- 위 예시는 샘플로 미리 들어가 있습니다. 편집·삭제 가능

## 자동으로 채워지는 것
| 탭 | 내용 |
|---|---|
| **통합시간표** | **3개 지점(대치·도곡·구룡초)을 한 시트에서 좌우로** 나란히 보는 정규 시간표. 시간 행 × (지점별 월~일) |
| **방학시간표_대치/_도곡/_구룡초** | **날짜별** 달력(행=날짜 7/24·7/25…, 열=시간 30분). 날짜·시간대별 강사 자동 표시 |
| **데일리현황_대치/_도곡** | 세로 날짜형 달력. 강사별 **그날 근무시간(h) 자동 계산** → 급여 연동 |
| **강사_급여** | 시급제/월급제만 입력 → 정규/방학시간·급여 자동 산출 |
| **대시보드** | 인건비·근무시간 + **월 매출 입력 시 인건비 비율·영업이익 자동** (40%↑ 빨강 경고) |

## 운영시간 / 방학 교시
- 정규: 월~금 12~21시 / 토 9~17시 / 일 9~14시 (도곡점 일요일 휴무)
- 방학: 1부 09:00~10:30 · 2부 10:30~12:00 · 3부 12:30~14:00 · 정규 14:00~19:30
  - 대치점 7/24~8/20, 도곡점 7/23~8/20 (2026년 기준)

## 동작 원리
- `입력`의 요일·시간을 읽어 각 시간표 칸에 `FILTER+TEXTJOIN`으로 해당 강사를 모두 모아 표시
- `데일리현황`은 날짜의 요일·정규/방학 여부에 맞춰 강사별 근무시간을 `SUMPRODUCT`로 자동 합산
- `급여`는 데일리현황을 정규/방학으로 나눠 합산 → 시급제는 ×시급, 월급제는 월급 고정

## 강사 추가 (10명 초과)
강사 정원은 코드 상단 한 줄로 정해집니다. 기본 **20명**입니다.
```js
강사수: 20,   // ← 이 숫자만 바꾸면 인원 증감 (예: 25, 30)
```
- **20명 이내**면 코드 수정 없이 `강사_급여` 탭의 빈 "강사명" 칸에 이름만 적으면 됩니다.
- **20명 초과**면 `강사수`를 늘리고 `[강사시간표] → [전체 다시 생성]` 한 번 실행.
- 재생성해도 기존 이름·시급·월급·입력 일정은 그대로 유지됩니다.
- (강사가 많지 않으면 `강사수`를 줄여 데일리현황의 빈 열을 줄일 수 있습니다.)

## 설정 변경 (코드 상단 CONFIG)
- `강사`: 강사 명단(=급여/현황 기준)
- `DAILY_START`/`DAILY_END`: 데일리 달력 기간
- `방학`: 지점별 방학 기간 / `교시`: 교시 시간
- `샘플`: 미리 채울 예시 입력

## 참고
- 시간표 칸은 입력에서 자동 계산되는 **수식**입니다. 특정 칸을 손으로 덮어쓰면 그 칸의 자동 연동은 끊깁니다. 일정 변경은 `입력` 탭에서 하세요.
- 한 시트로 자동 생성하지 못하는 환경을 위해 단순 양식 CSV(`강사시간표_통합.csv`)도 함께 보관되어 있습니다.
