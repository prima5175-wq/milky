# 강사 시간표 & 데일리 강사현황 (대치점 · 도곡점)

`강사시간표_통합.csv` — 구글시트로 가져오면(Import) 바로 사용할 수 있는 통합 양식입니다.
한 시트 안에 아래 구역이 순서대로 들어 있습니다. (강사1~강사10 임시명, 빈 양식)

## 운영시간 기준
- **정규**: 월·화·수·목·금 12:00~21:00 / 토 09:00~17:00 / 일 09:00~14:00
  - 도곡점은 **일요일 운영 안 함**
- **방학 특강**: 대치점 7/24~8/20, 도곡점 7/23~8/20 (2026년 기준)
  - 1부 09:00~10:30 / 2부 10:30~12:00 / 3부 12:30~14:00 / 정규 14:00~19:30

## 시트 구성(구역)
1. 사용 안내
2. 구역1. 강사 명단 (강사1~10)
3. 정규 주간 시간표 - 대치점 (시간 × 요일, `-`는 휴무)
4. 정규 주간 시간표 - 도곡점 (일요일 전체 휴무)
5. 강사별 정규 출근 설정 - 대치점 / 도곡점 (요일별 출근시간 기입)
6. 정규 데일리 강사현황 - 대치점 / 도곡점 (날짜 × 강사1~10)
7. 방학 데일리 강사현황 - 대치점 (7/24~8/20, 교시별)
8. 방학 데일리 강사현황 - 도곡점 (7/23~8/20, 교시별)

## 구글시트로 가져오는 법
1. Google Drive → 새로 만들기 → Google 스프레드시트
2. 파일 → 가져오기 → 업로드 → `강사시간표_통합.csv` 선택
3. 가져오기 위치: "새 시트 만들기" 또는 "현재 시트 바꾸기"
4. 구분 기호: 쉼표(,) 자동 인식

## 추천 추가 설정(선택)
- **조건부 서식**: `-`(휴무) 셀 회색 처리, 빈 운영셀 강조
- **데일리 현황 자동 집계**: 날짜 행 끝에 `=COUNTA(C행:L행)` 으로 그날 출근 강사 수 표시
- 강사별 색상 지정으로 한눈에 구분
