# 강사 시간표 · 데일리 강사현황 · 급여 · 경영 대시보드 (대치점 · 도곡점)

학원 운영을 한 파일에서 관리하는 Google Sheets 시스템입니다.
`AppsScript_시간표.gs` 한 개를 붙여넣고 실행하면 모든 탭이 **서식·색상·드롭다운·수식·급여연동·경영지표까지** 자동으로 만들어집니다.

## 설치 (3분)
1. 새 Google 스프레드시트 생성
2. 상단 메뉴 **확장 프로그램 → Apps Script**
3. 기존 코드 모두 지우고 `AppsScript_시간표.gs` 전체 붙여넣기 → 저장
4. 함수 목록에서 **buildAll** 선택 → ▶ 실행 → 최초 1회 권한 허용
5. 스프레드시트로 돌아오면 완성. 이후엔 시트 상단 **[강사시간표] → [전체 다시 생성]** 메뉴로 재생성 가능

## 생성되는 탭
| 탭 | 내용 |
|---|---|
| **대시보드** | 인건비·근무시간·지점지표 한눈에. **월 매출만 입력하면 인건비 비율·영업이익 자동 계산** (인건비율 40%↑ 빨강 / 30%↓ 초록 경고) |
| **강사_급여** | 강사1~10, 급여형태(**시급제/월급제** 드롭다운), 시급·월급 입력 → 급여 자동산출 |
| **정규시간표_대치 / _도곡** | 시간×요일 달력. 셀에서 강사 드롭다운 선택. 휴무 회색 처리 (도곡 일요일 휴무) |
| **방학시간표_대치(7/24~8/20) / _도곡(7/23~8/20)** | 날짜×교시(1·2·3부/정규) 배정 → 강사별 방학시간 자동집계 |
| **데일리현황_대치 / _도곡** | 세로 날짜형 달력. 주말·오늘 색상 강조, 강사별 근무시간(h) 입력 → 급여 연동 |

## 운영시간 기준
- **정규**: 월~금 12:00~21:00 / 토 09:00~17:00 / 일 09:00~14:00 (도곡점 일요일 휴무)
- **방학 특강**: 1부 09:00~10:30 · 2부 10:30~12:00 · 3부 12:30~14:00 · 정규 14:00~19:30
  - 대치점 7/24~8/20, 도곡점 7/23~8/20 (2026년 기준)

## 급여 연동 원리
```
급여 = (월급제) 월급 고정
     = (시급제) (정규시간 + 방학시간) × 시급
  · 정규시간 = 데일리현황 두 지점의 강사 근무시간 합계
  · 방학시간 = 방학시간표 교시 배정 × 교시 길이(1·2·3부 1.5h, 정규 5.5h)
```
강사명을 `강사_급여` 탭에서 실제 이름으로 바꾸면 모든 시간표·달력 헤더에 자동 반영됩니다.

## 설정 변경 (코드 상단 CONFIG)
- `강사`: 강사 이름/인원
- `DAILY_START` / `DAILY_END`: 데일리 달력 기간
- `방학`: 지점별 방학 기간
- `교시`: 교시 시간/길이 → 급여 자동집계에 반영

## 상업적 활용 포인트
- **인건비 실시간 파악**: 시급/월급만 넣으면 월 인건비 자동 집계
- **손익 관리**: 매출 입력 → 인건비 비율·영업이익 즉시 확인, 위험구간 색상 경고
- **지점 비교**: 대치/도곡 근무시간·인건비 분리 집계로 지점별 효율 비교
- **확장 용이**: 지점·교시·기간을 CONFIG로 조정해 분원 확장에도 그대로 재사용
