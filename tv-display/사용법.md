# 학원 하원 관리 TV 디스플레이 사용법

## 1. 구글 시트 준비

수강생 대장 구글 시트에 다음 열이 필요합니다:

| 열 | 내용 | 예시 |
|---|---|---|
| B | 학생 이름 | 김민수 |
| E | 하원 시간 | 15:30 |
| F | 예외사항/메모 | 엄마가 데리러 옴 |

> 열 위치는 설정 화면에서 변경 가능합니다.

## 2. 구글 시트 웹에 게시

1. 구글 시트 열기
2. **파일** → **공유** → **웹에 게시**
3. "전체 문서" 선택 → **게시** 클릭
4. 시트 URL에서 시트 ID 복사
   - URL: `https://docs.google.com/spreadsheets/d/여기가시트ID/edit`

## 3. TV에서 열기

1. TV에 연결된 PC/크롬캐스트에서 `index.html` 파일을 크롬 브라우저로 열기
2. 시트 ID와 열 설정 입력 → "연결 시작" 클릭
3. 설정은 브라우저에 저장되므로 다음번에는 자동 연결됩니다

## 4. 화면 구성

- **왼쪽**: 디지털 시계 (현재 시간)
- **오른쪽**: 하원 현황
  - 🔴 하원시간 지남 (몇 분 지났는지 표시)
  - 🟠 하원시간 (지금 하원)
  - 🟡 3분 전
  - 🔵 5분 전
  - 🟣 예외사항 (메모가 있는 학생)

## 5. 기타

- 데이터는 **30초마다** 자동 갱신됩니다
- 설정을 다시 열려면 화면을 **더블클릭**하세요
- 전체화면: `F11` 키
