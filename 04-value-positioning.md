# BookSpec Value Positioning

## 포지셔닝 문장

> **BookSpec은 독서논술학원에 자녀를 보내는 초·중등 학부모가 "돈은 내는데 아이가 뭘 하는지 모르는" 상황에서, 종이 독서록과 카카오톡 단톡방에 의존하던 기존 방식보다 — 학부모가 직접 보상을 설계하고, AI가 독서 기록을 세특 포트폴리오로 연결해주는 방식으로 — 아이의 독서 동기와 입시 성과를 동시에 관리하는 문제를 더 잘 해결한다.**

---

## 구성 요소 분해

| 구성 요소 | 내용 |
|-----------|------|
| 타겟 고객 | 독서논술학원에 자녀를 보내는 초·중등 학부모 |
| 이 상황 | 돈은 내는데 아이 독서 현황을 모르고 보상도 직접 못 주는 상황 |
| 기존 방식 | 종이 독서록 + 카카오톡 단톡방 |
| 이렇게 더 | 학부모가 직접 보상을 설계하고, AI가 세특 포트폴리오로 연결 |
| 핵심 니즈 | 아이의 독서 동기 + 입시 성과를 동시에 관리 |

---

## 핵심 장력

"독서 동기와 입시 성과를 **동시에**" — 기존 서비스들이 둘 중 하나만 건드리는 게 공백.

- 게임화 앱(북적북적 등) → 동기만, 입시 연결 없음
- 교육부 독서로 → 입시 기록만, 동기부여 없음
- BookSpec → **둘 다**
